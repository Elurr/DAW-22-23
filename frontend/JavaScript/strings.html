Strings con JS

Un string es cero o más caracteres escritos dentro de comillas.

Esto lo convierte básicamente en una cadena de caracteres.

Se pueden usar tanto comillas simples como dobles. Un string pueden contener comillas siempre y cuando sean distintas de las que rodean el string:

let str = 'Tere "Terere" Hererra'

Si imprimimos este string: Tere "Terere" Hererra.

Métodos de strings

length

Para obtener la longitud de string se usa el método length().

let txt = 'ABCDEFGHIJKLMNOPQRSTUVWXYZ'
let longitud = txt.length;

slice()

Corta o extrae una parte de un string y la retorna como un nuevo string.

slice() necesita dos parámetros. La posición inicial y la final (no incluída en el corte).

substr()

Funciona similar a slice(), pero el segundo parametro indica la longitud del string a recortar. Deprecado.

replace()

Reemplaza una parte de un string y lo retorna como un string nuevo.

El primer parámetro es el valor a buscar en el string y el segundo es el valor por el que lo reemplazamos.

Estos valores son case sensitive. Es decir, que los valores de mayúsculas y minúsculas deben coincidir.

Sólo se reemplaza la primera coincidencia en caso de haber varias en el string.

Para sortear la diferencia de las mayúsculas, se puede usar una expresión regular.

Transformar a minúsculas y viceversa

Podemos usar toLowerCase() o toUpperCase() para transformar un string o una parte en minúsculas o mayúsculas, respectivamente.

Limpiar strings

Podemos usar trim() para limpiar los espacios en blanco redundantes de un string.

Si tenemos un str nombre = " Román" el trim lo dejará como "Román".

charAt()

Retorna el carácter ubicado en el índice especificado como parámetro.

Convertir string en arrays con split()

Si tenemos un string cuyos valores están separados por comas o cualquier otro caracter distinguible, podemos servirnos de esto para transformarlo en un array.
